## 파이썬 코드

python -m venv venv
파이썬이라는 프로그램에서 모듈을 만들건데 그 모듈은 venv이고 파일 이름은 venv 야
> 그러면 파일이 여러개 생성 
> 이 파일은 무엇? 가상환경에서만 쓰게 하는 거 밖으로 못 내보냄

 source venv/Scripts/activate
 > venv/Scripts/activate 라는 파일을 활성화할거야
